## 二、衍生指标计算

### 2.1 毛利率

```
毛利率 = gp / revt

FY2023：2228.67 / 5160.10 = 43.2%
FY2024：1944.73 / 4707.69 = 41.3%
FY2025：2444.52 / 5810.12 = 42.1%
```

**结论：** 毛利率稳定在 41–43%，未因整合或高速产品切换而恶化。

### 2.2 营业利润率（oibdp margin）

```
FY2023：1084.37 / 5160.10 = 21.0%
FY2024：852.64 / 4707.69  = 18.1%
FY2025：1143.78 / 5810.12 = 19.7%
```

**结论：** FY2024整合期利润率下滑，FY2025已回升，方向向好。

### 2.3 营收增速（合并后可比段）

```
FY2024 vs FY2023：4707.69 / 5160.10 - 1 = -8.8%（整合阵痛，一次性）
FY2025 vs FY2024：5810.12 / 4707.69 - 1 = +23.4%（强势反弹）
```

### 2.4 历史 P/S（用财年末数据）

```
市值 = prcc_f × csho（单位换算：csho百万股 → 亿股除以100）

FY2023：市值 = 50.98 × 1.3958亿 = 71.2亿  →  P/S = 71.2 / 516.0 = 1.38x
FY2024：市值 = 72.46 × 1.5278亿 = 110.7亿 →  P/S = 110.7 / 470.8 = 2.35x
FY2025：市值 = 89.21 × 1.5556亿 = 138.8亿 →  P/S = 138.8 / 581.0 = 2.39x
```

**关键发现：** FY2025年末（2025年6月）P/S只有 2.4x。到2026年5月股价涨至约$365，P/S膨胀至约9.8x。这一年的涨幅主要是**估值倍数扩张**，而非业绩增长驱动。

### 2.5 当前 P/S（2026-05-27，已修正）

> ⚠️ v1.0使用了WRDS的FY2025流通股数155.56M，已修正为IBKR最新值195.64M。

```
【修正后计算】
当前市值 = $378.86 × 1.9564亿股 = 741亿

P/S on TTM营收（$6.61B）= 741 / 661 = 11.2x  ← 与IBKR显示11.3x吻合
P/S on FY2025年度营收（$5.81B）= 741 / 581 = 12.8x

【v1.0错误计算（仅供参考）】
当时用 $365 × 1.5556亿 = 568亿，P/S = 9.8x → 低估了约15%
原因：FY2025至今公司增发了约4000万股（155.56M → 195.64M）
```

**修正后结论：** 当前P/S实际为**11.2x（TTM）**，比之前估算的9.8x更高。Morningstar和IBKR显示的11.3x与此吻合。

## 三、目标价推导

### 核心逻辑（反推法）

```
目标股价 = (目标P/S × 预期营收) / 流通股数
```

**输入假设（v1.3修正版，基于管理层已确认数据）：**

> v1.2 → v1.3 方法修正：原模型用 "TTM × 1.20" 作为 FY2026E，此方法混淆了TTM周期与财年。
> 正确做法：用管理层确认的FY2026E（$7.05B）为基础，推算FY2027E目标，以Forward P/S定价。

- **FY2026E营收（已确认）：$7,053M ≈ $7.05B**（YTD $5,073M + Q4指引中位数 $1,980M）
- **FY2027E营收（假设+20%）：$7,053M × 1.20 = $8,464M ≈ $8.46B**
- 流通股数：**1.9564亿股**（IBKR，as of 2026-05-04）

| 场景 | 目标P/S（Forward on FY27E）| FY2027E营收 | 目标市值 | **目标股价** | v1.2变动 |
|------|--------------------------|-----------|---------|-----------|---------|
| 保守 | 9x | $8.46B | $762亿 | **$389** ← 略高于现价 | $365 → +$24 |
| 基准 | 11x | $8.46B | $931亿 | **$476** | $446 → +$30 |
| 乐观 | 13x | $8.46B | $1,100亿 | **$562** | $565 → -$3 |

> **关键变化：** 保守场景目标价从$365上调至$389，现已略高于当前股价$378，说明在FY2027增速可持续20%的假设下，当前价格具有一定的安全边际。基准场景$476对应的催化剂是FQ4'26业绩确认 + 市场对FY2027的增速预期不降。

**止损对应：** $350 ≈ 当前P/S略有压缩 + 增速放缓场景，是设定的操作止损线，不是基本面底线。若以基本面定止损，8x P/S on TTM $6.61B / 1.9564亿股 = **$270**（Morningstar公允价值$239.77附近）。

> 两个止损逻辑：
> - **操作止损$350**：技术面 + 情绪保护，用于日常持仓管理
> - **基本面底线~$270**：对应无AI溢价时的合理估值，触及则说明行情整体结束

## 四、模型不确定性（重要）

这个模型有四个主观判断点，每一个都会影响最终目标价：

### ① 增速20%从哪里来？【已被数据确认】

~~这里用的20%是假设示例，不是算出来的。~~ 

**v1.2更新：已下载WRDS Compustat Quarterly数据，20%假设已验证。**

实际FY2026前三季增速：+17.3%、+17.5%、+20.5%，均值+18.4%，且趋势正在重新加速。20%作为全年预测依据合理，甚至略保守。

**已确认（v1.3）：**
- FY2026E营收：$7.05B（YTD $5.07B + Q4指引中位数$1.98B），YoY +21.4%
- FY2027E（20%增速）：**$8.46B** → 这是目标价计算的基础
- Q4指引显示增速正在加速（+25-34% YoY），20%假设用于FY2027是保守的

### ② 股价用的是某一天的点位，不是均价
专业分析师通常用**30天或90天均价**计算当前P/S，避免单日波动干扰。
今天股价$365，几天前是$385，P/S可以差出10%。这个误差是正常的，目标价应当给出区间，不给单点。

### ③ 流通股数应使用最新值【已修正】
FY2025的155.56M只是到2025年6月的数据。IBKR显示截至2026年5月4日，最新流通股数为**195,639,321股**，较FY2025增加了约4000万股（+25.8%），大概率是可转债转股或定向增发导致。

**这一增发对目标价的影响：**
- 相同市值下，每股价格被摊薄约20%
- v1.0的目标价$448（10x P/S）在修正股数后对应约$365，已等于当前价格
- 若继续持有并期待$450目标，需要P/S维持11x以上

最新股数查询方式：
- IBKR → Fundamentals → Company Profile → Common Shares Outstanding
- SEC官网 → 最新10-Q / 10-K（每季度更新）

### ④ 目标P/S = 10x 是判断，不是精确值
这一步没有公式，只有参考依据：
- COHR自身历史：FY2025末P/S为2.4x，当前9.8x已处于历史高位
- 同行参考：AI行情中光通信板块整体估值抬升，10x处于合理但非极端位置
- 增速支撑：20%增速对应10x P/S是市场经验值，增速掉到10%则10x难以维持

**结论：** 目标价$450–$480是在上述四个假设下成立的参考值，假设变化目标价随之变化。投资决策不能只看这个数字，要持续跟踪实际季度增速是否兑现。
